# Support and Tools

Get the status of the Meta Developer Platform APIs, explore available Meta Developer Tools, and find support resources.

---

## Meta Developer Site Navigation

At the top of all pages on the [Meta for Developers](https://developers.facebook.com/) site, you will find these key tabs:

- **Apps**: The App Dashboard to configure your app's APIs, products, and SDKs.
- **Docs**: Comprehensive implementation guides and references.
- **Required Actions**: Critical maintenance tasks (e.g., Data Use Checkup) needed to retain API access.
- **Support**: Resources for reporting bugs, asking questions, and monitoring platform status.
- **Tools**: Testing, debugging, and previewing utilities.

## Messaging Status Metrics

For users of the **Instagram Messaging API**, specific availability and latency metrics are available:

- **Availability**: Calculated as a percentage of uptime over a measured period (e.g., 99.9%).
- **Latency (P90/P99)**:
    - **Inbound**: Time from user message to Meta making the first Webhook attempt.
    - **Outbound**: Time from Meta receiving your send request to being ready for transmission.
